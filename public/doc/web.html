Web

A web is a set of lines drawn between given points, in such a way that intersections do not occur.

As customary, webs will be introduced via annotated code, which generates this image:

Here's the code:

import {rs as linePP} from '/shape/line.mjs'; import {rs as basicP} from '/generators/basics.mjs'; import {rs as addPointMethods} from '/mlib/pointGen.mjs'; import {rs as addWebMethods} from '/mlib/web.mjs'; let rs = basicP.instantiate(); addPointMethods(rs); addWebMethods(rs); rs.setName('web_wheel'); let rd= 3000; let topParams = {webTries:1000,numRings:20,radius:rd,numPointsPerRing:20,framePadding:1.2*rd,minConnectorLength:0,maxConnectorLength:2000} Object.assign(rs,topParams); rs.initProtos = function () { let lineP = this.set('lineP',linePP.instantiate()).hide(); this.lineP.stroke = 'white'; this.lineP['stroke-width'] = 5; } rs.initialize = function () { this.initProtos(); let pnts = this.genRings(this); this.generateWeb({points:pnts}); this.addFrame(); } export {rs};

A web is a set of lines ("connectors") obeying the length constraints which connect given points without intersecting. generateWeb successively adds new connectors until it impossible to add a connector without intersection. generateWeb has this type: generateWeb({points:arrayOf(Point),minConnectorLength:number,maxConnectorLength,lineP:line});

generateWeb will take its parameters from this if they are not found in the parameter object.
