First have a look at some images that this JavaScript library has generated

PrototypeJungle runs in the browser, and can be downloaded here, or cloned from github. It comes with its own little web server, written in node, intended for running locally, so that you can generate and save your own images. Installation takes a minute. The software is free and open source (MIT license).
